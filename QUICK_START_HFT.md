# 🎮 QUICK START: HFT MODE

## ⚡ 30-Second Setup

1. **Open the app**: http://localhost:8000
2. **Login**: Click Demo Mode (fastest)
3. **Add bots**: Click `+ ADD BOT` button (add 3-12 bots)
4. **Launch HFT**: Click `🚀 HFT START` button
5. **Watch**: Monitor `TRADES/MIN` counter in header

## 🎯 Key Features

| Feature | What It Does |
|---------|------------|
| **🚀 HFT START** | Activate all bots for parallel trading |
| **🛑 HFT STOP** | Stop all bots instantly |
| **TRADES/MIN** | Real-time trading velocity (60-sec window) |
| **Trade Log** | All executed trades with P&L |
| **Balance** | Updated in real-time |

## 📊 Performance Expectations

- **With 3 bots**: 150-450 trades/min
- **With 6 bots**: 300-900 trades/min  
- **With 12 bots**: 600-1800 trades/min

Each bot trades every 400-1200ms with AI decisions.

## 🔧 Advanced Controls

| Button | Purpose |
|--------|---------|
| `⏹️ STOP ALL` | Pause all bots (keeps them ready) |
| `▶️ PLAY ALL` | Resume paused bots |
| `AUTO` (per bot) | Toggle individual bot auto-trade |
| `🎰 SPIN` (per bot) | Manual single trade |

## 💡 Tips for Best Results

2. **Monitor the agent audit panel** - Watch agent performance
3. **Check trade ledger** - See which strategies are winning
4. **Use batch mode for speed** - 🚀 START launches everything at once
5. **Mix bot strategies** - Different strategies = better diversification

## 🚨 Troubleshooting

**No trades appearing?**
- Make sure at least 1 bot is added
- Click 🚀 HFT START (not individual AUTO buttons)
- Check browser console (F12) for errors

**TPM not updating?**
- Refresh the page (Ctrl+R)
- Make sure bots are spinning (SPIN button working)
- Check API connection to CoinGecko

**Bots not trading?**
- Set a bet amount on the bot
- Click 🎰 SPIN manually first (tests API)
- Then click 🚀 HFT START for batch mode

## 📈 What's Happening

When you click `🚀 HFT START`:
1. All bots activate simultaneously
2. Each bot independently polls market data
3. Claude AI analyzes and makes decisions
4. Trades execute with simulated outcomes
5. P&L updates in real-time
6. Every trade adds to TRADES/MIN counter
7. Trade log fills with activity

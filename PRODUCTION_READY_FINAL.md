## 🛡️ RISK MITIGATION & COOLDOWN LIMITS

To protect live capital and ensure conservative transactional authority, the platform defaults to the maximum risk constraints and safe cooldown gaps:
- **Loss Trigger**: `15` consecutive combined losses triggers circuit breakers.
- **Level 1 Cooldown**: `30` minutes of absolute autotrading suspension.
- **Level 2 Cooldown**: `2` hours of absolute autotrading suspension.
- **Level 3 Cooldown**: `48` hours of complete platform shutdown.
- **Enforced Stop-Loss**: Every bot trade enforces a strict 3:1 reward-to-risk ratio ($10 maximum loss, $30 take profit).

## 🔑 SECURE LOCAL KEYSTORE IMPORT

To securely import your live mainnet account private key into an encrypted local keystore on your machine without leaking credentials to plain text files, use Foundry's interactive `cast` command:

```bash
# Securely import account (opens a hidden prompt for key + password)
cast wallet import mainnet_deployer --interactive
```

To run a deployment using this imported account with no keys in plain text:
```bash
forge create src/PayoutManager.sol:PayoutManager --account mainnet_deployer --broadcast
```

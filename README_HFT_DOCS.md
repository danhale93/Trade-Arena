### Common Issues & Solutions

**"TPM counter not updating"**
- Check: Bots have bet amounts set
- Try: Click 🎰 SPIN manually first
- Fix: Refresh page (Ctrl+R)

**"No trades appearing"**
- Check: At least 1 bot is added
- Try: Click 🚀 HFT START button
- Fix: Check browser console (F12) for errors

**"Server not responding"**
- Run: `python -m http.server 8000`
- Check: Port 8000 is not in use
- Try: Different port (8001, 8002)

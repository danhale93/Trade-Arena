## 🐛 IF SOMETHING WRONG

### Master Switch Not Working
```
1. Refresh page (Ctrl+F5)
2. Try clicking directly on button
3. Try Ctrl+Space shortcut
4. Check browser console (F12)
5. Look for red error messages
```

### Balance Not Updating
```
1. Wait 1-2 seconds (updates every 1 sec)
2. Make sure page is active (not minimized)
3. Run a trade to trigger movement
4. Refresh page if stuck
5. Check console for errors
```

### Bots Not Spinning
```
1. Check Master ✓ ON (green)
2. Check no emergency stop (red button)
3. Check no "HIGH GAS" message
4. Check no "DRAWDOWN KILL" message
5. Try clicking SPIN manually
```

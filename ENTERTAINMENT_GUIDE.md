## 🎬 CUSTOMIZATION OPTIONS

Want to modify it? Here's what you can customize:

### Change Commentary:
Edit `crucible-entertainment.js` line 35-93:
```javascript
commentary: {
  wins: [
    "YOUR CUSTOM TEXT HERE!"
    // Add your own funny comments
  ]
}
```

### Adjust Sounds:
Change URLs in `crucible-entertainment.js` line 15-24:
```javascript
sounds: {
  win: 'YOUR_SOUND_URL_HERE',
  // Replace with your audio files
}
```

### Modify Colors:
Edit CSS in `createUIContainer()` method:
```css
.commentary-box.win {
  background: linear-gradient(...);  /* Change colors */
  border-color: #00ff88;              /* Change border */
}
```

### Adjust Animation Speed:
Edit animation durations in CSS:
```css
animation: slideIn 0.5s ease-out;  /* Change 0.5s to faster/slower */
```

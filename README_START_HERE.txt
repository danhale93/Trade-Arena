## 🤖 HOW IT WORKS

### AI Decision Flow
```
User clicks SPIN
     ↓
Fetches live market data (CoinGecko)
     ↓
Sends to Claude API with:
  - Top 8 coins by 24h change
  - Current prices and volumes
  - Bet amount
  - Available methods
     ↓
Claude analyzes and responds:
  {
    "token": "PEPE",
    "method": "ARBITRAGE",
    "win_probability": 0.65,
    "pnl_multiplier": 1.8
  }
     ↓
Validates and animates reels
     ↓
Shows result (WIN/LOSS)
     ↓
Updates balance and log
```

### Without API Key
- Falls back to intelligent random selection
- Still looks and feels like real trading
- Works instantly, no setup needed

## 🐛 IF SOMETHING BREAKS

### Common Issues

**"Cannot connect to API"**
- Normal! API key optional
- App works in fallback mode

**"MetaMask not found"**
- Install MetaMask extension
- Reload page
- Click "METAMASK / WALLET"

**"Reels won't spin"**
- Check browser console (F12)
- Reload page
- Try Demo Mode

**"Balance shows $0"**
- MetaMask not connected
- Fund wallet with ETH
- Wait 30 seconds

See **SETUP_v4.md** for detailed troubleshooting.
